## ğŸ¨ Design System Guidelines

### Color Palette


### Typography Scale

- **Headings**: 2.5rem, 2rem, 1.5rem, 1.25rem

- **Line Height**: 1.5 for body, 1.2 for headings

- **Scale**: 0.25, 0.5, 1, 1.5, 2, 3, 4, 6, 8, 12, 16, 20, 24
